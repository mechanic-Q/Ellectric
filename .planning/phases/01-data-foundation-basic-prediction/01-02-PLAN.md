    Function `train_model(df, target_col="load_mw", n_splits=5, gap=24)`:
    - Accepts a DataFrame with feature columns + target column (from features.parquet)
    - Separates X = df.drop(columns=[target_col]), y = df[target_col]
    - Creates `TimeSeriesSplit(n_splits=n_splits, gap=gap)` — gap=24 prevents autocorrelation leakage from lag-24h
    - For each fold (train_idx, val_idx):
      - X_train, X_val = X.iloc[train_idx], X.iloc[val_idx]
      - y_train, y_val = y.iloc[train_idx], y.iloc[val_idx]
      - **CRITICAL**: Create NEW scaler inside fold: `scaler = StandardScaler()`
      - `scaler.fit(X_train)` — train data ONLY, never X_val or full X
      - `X_train_s = scaler.transform(X_train)`
      - `X_val_s = scaler.transform(X_val)` — apply, do NOT re-fit
      - Create `xgb.XGBRegressor(n_estimators=100, max_depth=6, learning_rate=0.1, random_state=42)`
      - `model.fit(X_train_s, y_train)`
      - `y_pred = model.predict(X_val_s)`
      - Compute metrics using sklearn: `mean_absolute_error`, `mean_squared_error` (sqrt for RMSE), `mean_absolute_percentage_error`, `r2_score`
      - Record fold metrics in a list of dicts: fold number, train_rows, val_rows, mae_mw (round 2), rmse_mw (round 2), mape_pct (round 2), r2 (round 4)
      - Track best model by lowest MAE
    - Return (best_model, pd.DataFrame(fold_metrics))
